### *Caregiver-Management-System*

---

Caregiver Management System created in Java

---

1. Designed and developed a production-ready Caregiver Management System using Java, MySQL, HTML, and CSS to streamline caregiver-patient assignments, user management, and secure data storage.
2. Built the application with Spring Boot and Hibernate, reducing development time by 50% while maintaining full efficiency.
3. Applied the Dependency Injection design pattern to enhance testability and maintainability.
4. Implemented the MVC architecture to improve scalability, readability, and extensibility.
5. Secured the application with Spring Security, preventing unauthorized access and protecting sensitive data.
